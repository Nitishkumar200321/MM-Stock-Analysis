# 📈 Mahindra & Mahindra Stock Analysis

## 🔍 Project Overview
Real-time stock data analysis of M&M (NSE: M&M.NS) 
using Python, Google Colab, and Looker Studio Dashboard.

## 🛠️ Tools & Technologies
- Python 3 (Google Colab)
- yfinance — real-time stock data
- Pandas, Numpy — data cleaning
- Plotly, Matplotlib — visualizations
- Looker Studio — interactive dashboard

## 📊 What I Analyzed
- 1 Year of M&M stock data (248 trading days)
- Technical Indicators: SMA, EMA, RSI, MACD, 
  Bollinger Bands
- Price trends, Volume analysis
- Up vs Down day distribution
